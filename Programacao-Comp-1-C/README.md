# Programação de Computadores I (Linguagem C)

Exercícios práticos focados em algoritmos fundamentais e gerenciamento de memória.

## Destaques do Código
Aqui estão as implementações que demonstram lógica avançada e uso de recursos de baixo nível:

### Gerenciamento de Memória & Estruturas
* **[Sistema de Cadastro Dinâmico](./alocacaodinamica.c):** Utiliza `malloc` para criar vetores de `structs` em tempo de execução, permitindo cadastrar N usuários sem desperdício de memória.
* **[Aritmética de Ponteiros](./ponteiros3.c):** Demonstração de navegação em matrizes e vetores utilizando apenas endereços de memória.

### Algoritmos & Lógica
* **[Busca de Substring Manual](./ponteiros2.c):** Algoritmo que percorre uma string via ponteiros para encontrar ocorrências de termos (sem usar `strstr`).
* **[Recursividade](./recursao.c):** Implementação de funções recursivas para contagem e somatórios.

**Nota:** Os arquivos contêm múltiplas soluções modulares para os problemas propostos nas listas acadêmicas.
